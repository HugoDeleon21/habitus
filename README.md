# Habitus

Habitus é uma aplicação web de rastreamento de hábitos desenvolvida como projeto da disciplina de Desenvolvimento de Sistemas.

## Objetivo

O objetivo do sistema é permitir que usuários cadastrem, acompanhem e gerenciem hábitos diários por meio de uma interface web integrada a uma API REST.

## Tecnologias previstas

- C# .NET Web API
- PostgreSQL
- Docker
- Entity Framework Core
- Swagger
- HTML, CSS e JavaScript
- xUnit para testes automatizados

## Funcionalidades principais

- Cadastro de usuário
- Login
- Cadastro de hábitos
- Visualização de hábitos por data
- Edição de hábitos
- Exclusão de hábitos
- Marcação de hábito como concluído
- Métricas simples de desempenho diário
